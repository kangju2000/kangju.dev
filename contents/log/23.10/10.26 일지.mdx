우아한테크코스 1주차가 끝나고 피어리뷰를 진행했다.

다른 분들이 리뷰 남겨주신 것들을 보고 다시 한번 생각해볼 수 있어서 좋았다.

새로 알았던 것 중 하나는 `Set`에 들어가는 값이 array만 되는 줄 알았는데 string도 들어갈 수 있다는 것이었다.

나는 기존 string을 split해서 array로 만들어서 set에 넣었는데, 그냥 string을 넣어도 된다는 것이다.

```js
const set = new Set('hello')

console.log(set) // Set(4) {"h", "e", "l", "o"}
```

내가 다른 분들 코드를 리뷰하면서 인상 깊었던 것 중 하나는, `jsDoc`을 사용해서 함수의 인자와 리턴값에 대한 설명을 적어주는 것이었다.
